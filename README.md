Python Alarm Clock 🕒
<br/>
<br/>
A progressive Python project that evolved from a simple single-alarm system to a multi-alarm application. 
<br/>
This repository showcases my learning journey in Python, moving from basic time handling to more complex logic management.

<br/>

  <br/>
• mono/ : <br/>
          Description ⇝ Initial Version that i made ; when i was learning the basics of time handling and usage of loops

          <br/>

                           playsound 3
                           os
